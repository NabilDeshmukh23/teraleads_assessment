DentalAI - MERN Stack Patient Management
A professional patient registry and dental assistant portal built with the MERN stack and integrated with Google Gemini AI for clinical decision support.

Key Features
AI Dental Assistant: A "Senior Dentist" chatbot that provides patient summaries and medical alerts.

Patient Registry: Full CRUD (Create, Read, Update, Delete) functionality with real-time search.

Secure Auth: Role-based authentication with persistent user profiles.

Database: Hosted on Neon PostgreSQL using Prisma ORM for type-safe queries.

 Setup Instructions
Clone the repo: git clone https://github.com/NabilDeshmukh23/teraleads_assessment.git

Install Dependencies: Run npm install in both /client and /server.

Environment Variables: Create a .env in /server with your DATABASE_URL and GEMINI_API_KEY.

Run the Project:

Server: node index.js

Client: npm run dev
